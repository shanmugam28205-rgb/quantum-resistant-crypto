Why migrate at all

Two algorithms, one deadline

Quantum computers don't threaten all cryptography equally. The danger is concentrated in exactly the algorithms the internet depends on for key exchange and signatures.

Shor's Algorithm breaks outright

Solves integer factorization and discrete logarithms in polynomial time on a quantum computer — the exact hard problems RSA, ECC, and Diffie-Hellman are built on.

→ RSA, ECC, DH: fully broken, not just weakened

Grover's Algorithm weakens only

Gives a quadratic brute-force speedup against symmetric ciphers and hash functions — serious, but solved by doubling key size, not by a redesign.

→ AES-128 → AES-256, SHA-256 → SHA-512
— public basis (messy, published) — private basis (clean, secret) —

The idea underneath ML-KEM & ML-DSA

Picture points arranged in a perfectly regular grid stretched across many dimensions — a lattice. There are countless ways to describe the same grid. A tidy, orthogonal set of directions makes it trivial to find the point nearest to any target. A tangled, skewed set describes the identical grid but makes that same search practically unsolvable.

The tidy description is the private key. The tangled one is published as the public key. No known algorithm — classical or quantum — can efficiently untangle one back into the other. That asymmetry is the entire security guarantee behind Kyber and Dilithium.

What's implemented

Three standards, three jobs

Each NIST-selected algorithm targets a different cryptographic role. This repo wraps all three via the audited liboqs reference implementation.

ML-KEMFIPS 203 · Kyber
Key encapsulation. Lattice-based (Module-LWE). Replaces RSA/ECDH for establishing a shared secret over an open channel.
Key exchange
ML-DSAFIPS 204 · Dilithium
Digital signatures. Lattice-based (Module-LWE/SIS). Replaces RSA/ECDSA — compact, fast to verify.
Signing & verification
SLH-DSAFIPS 205 · SPHINCS+
Digital signatures. Hash-based only — no lattice assumption. Larger signatures, but the most conservative security proof of the three.
Backup signing scheme
